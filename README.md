## 🚀 How to Use

1. Clone the repository:

```bash
git clone https://github.com/Abhi042004/LeetCode-Solutions.git
```

2. Navigate to the required topic and difficulty folder.

3. Compile the Java file:

```bash
javac FileName.java
```

4. Run the program:

```bash
java FileName
```
